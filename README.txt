Để chương trình có thể chạy, cần có SQL Server chạy tại cổng 1433
Cần phải cài đặt môi trường DOTNET SDK 6.0
Cần cài đặt bộ entity framework core và entity framework core design
Mở Terminal tại thư mục của chương trình, chạy lệnh "dotnet ef database update" để migration Schemas của Database
Gõ "dotnet run" để chạy chương trình
Chương trình sẽ lắng nghe ở cổng 5000
